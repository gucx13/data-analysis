安装Geant4和ROOT（Ubuntu16.04）

这里介绍如何在Ubuntu16.04上安装Geant4和ROOT。

重要提醒

  • 使用默认的bash
  • 用户名为geant4

  • 以Geant4 10.00.p02和ROOT v5.34.19为例，假设Geant4和ROOT安装至

      /projects/soft/ext

  • 需要预安装的软件多数直接用apt-get install或synaptic安装，但Pythia6安装至

      /projects/soft/ext/pythia6

预备软件的安装

Ubuntu安装后，建议先安装synaptic：

sudo apt-get update
sudo apt-get install synaptic

然后打开synaptic，在Settings->Repositories->Download from:中选择Other...，从中选择合适的源。教育网一般可以选择科大的镜像站点http://mirrors.ustc.edu.cn/ubuntu。

更换软件源后不要忘记Reload。

对于新手来说，利用synaptic查找并安装软件更为简单。

安装ROOT

准备源文件

 sudo mkdir -p /projects/soft/ext/sources
 cd /projects/soft
 sudo chown -R geant4:geant4 ext
 （注意，两个geant4分别表示用户名和组名，如果你用了别的用户名，需要进行相应的替换。你可以在命令行执行`echo $USER`和`echo $GROUP`得到当前的用户名和组名。）
 mkdir -p ext/root/pro
 cd ext/sources
 wget ftp://root.cern.ch/root/root_v5.34.19.source.tar.gz
 tar -zxvf root_v5.34.19.source.tar.gz

环境变量设置

编辑文件$HOME/ROOT.sh，内容为

 #!/bin/bash
 #

 export ROOTSYS=/projects/soft/ext/root/pro
 if ! echo ${PATH} | grep -q $ROOTSYS/bin ; then
    export PATH=$ROOTSYS/bin:$PATH
 fi

 if [ -n "$LD_LIBRARY_PATH" ]; then
    if ! echo ${LD_LIBRARY_PATH} | grep -q $ROOTSYS/lib ; then
       export LD_LIBRARY_PATH=$ROOTSYS/lib:$LD_LIBRARY_PATH
    fi
 else
    export LD_LIBRARY_PATH=$ROOTSYS/lib
 fi

通过source $HOME/ROOT.sh设置环境变量ROOTSYS，并更新PATH和LD_LIBRARY_PATH。

准备安装脚本

编辑脚本/projects/soft/ext/sources/install_root.sh，包含如下内容：

 #!/bin/bash

cd /projects/soft/ext/sources/root
source $HOME/ROOT.sh

./configure --fail-on-missing \
   --enable-builtin-pcre \
   --enable-builtin-freetype \
   --enable-cintex \
   --enable-explicitlink \
   --enable-gdml \
   --enable-genvector \
   --enable-mathmore \
   --enable-minuit2 \
   --enable-mysql \
   --enable-python \
   --enable-qt \
   --enable-qtgsi \
   --enable-reflex \
   --enable-roofit \
   --enable-table \
   --enable-unuran \
   --with-pythia8-incdir=/usr/include/Pythia8 \
   >& log.configure

 make -j2 >& log.make
 make install >& log.install

编译安装

cd /projects/soft/ext/sources
source install_root.sh

安装Geant4.10.00.p02

准备源文件

 cd /projects/soft/ext/sources
 <download geant4.10.00.p02.tar.gz>
 tar -zxvf geant4.10.00.p02.tar.gz
 cd /projects/soft/ext
 mkdir G4data
 cd G4data
 <download data files>

这些数据文件包括：

  • G4ABLA.3.0.tar.gz
  • G4NEUTRONXS.1.4.tar.gz
  • G4SAIDDATA.1.1.tar.gz
  • G4EMLOW.6.35.tar.gz
  • G4PhotonEvaporation.3.0.tar.gz
  • RealSurface.1.0.tar.gz
  • G4ENSDFSTATE.1.0.tar.gz
  • G4PII.1.3.tar.gz
  • G4NDL.4.4.tar.gz
  • G4RadioactiveDecay.4.0.tar.gz

解压缩这些文件：

 for i in `ls *.gz` ; do tar -zxvf $i ; done

编译安装

    cd /projects/soft/ext/sources
    mkdir build-geant4.10.00.p02
    cd build-geant4.10.00.p02
    cmake -D CMAKE_INSTALL_PREFIX=/projects/soft/ext/geant4.10.00.p02-install -DGEANT4_INSTALL_DATADIR=/projects/soft/ext/G4data -DGEANT4_BUILD_MULTITHREADED=ON -DGEANT4_USE_GDML=ON -DGEANT4_USE_QT=ON -DGEANT4_USE_XM=ON -DGEANT4_USE_OPENGL_X11=ON -DGEANT4_USE_INVENTOR=OFF -DGEANT4_USE_RAYTRACER_X11=ON -DGEANT4_USE_SYSTEM_ZLIB=ON ../geant4.10.00.p02    
    make -j2
    make install

这里，我们通过cmake指定：

  • Geant4安装目录为/projects/soft/ext/geant4.10.00.p02-install
  • 数据文件目录为/projects/soft/ext/G4data
  • 启用多线程模式
  • 启用GDML，QT，XM，OPENGL_X11，RAYTRACER_X11，ZLIB
  • 禁用INVENTOR

环境设置

source /projects/soft/ext/geant4.10.00.p02-install/bin/geant4.sh

编译例子

    mkdir -p $HOME/workdir/geant4
    cd $HOME/workdir/geant4
    cp -r /projects/soft/ext/geant4.10.00.p02-install/share/Geant4-10.0.2/examples .

    mkdir build-B1
    cd build-B1
    cmake ../examples/basic/B1
    make
    ./exampleB1
